# UI

## Theme

Teal accent palette. No gradients. Semantic tokens: accent.solid, accent.subtle, accent.fg.

## Keybinds

Chakra `Kbd` component. `Keybind` wrapper shows **⌘** on Mac, **Ctrl** on Windows/Linux for `mod` key. Used in Footer and TodoForm.

## Visual

Solid accent on header icon. Priority icons (↑/−/↓). Subtle shadows on panels. Light/dark via next-themes.

## Animations

Motion: float on empty state, check-pop on complete.
