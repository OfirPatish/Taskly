# Components

## Todo

| Component | Role |
| --------- | ---- |
| TodoForm | Textarea, priority icon, add — Enter to add, Shift+Enter new line |
| TodoItem | Checkbox, text, priority icon, edit, delete — double-click to edit |
| TodoList | List, empty state, delete confirm dialog |
| TodoFilters | Search, All/Active/Done pills, sort toggle |
| EmptyState | Contextual message + float animation |

## Layout

| Component | Role |
| --------- | ---- |
| Header | Logo, stats (Total/Active/Done), progress ring, color mode |
| Footer | Task count, mod+K hint, clear completed |
| TodoPage | Main page composition |

## UI

Provider, Toaster, ColorModeButton, Keybind (platform-aware: ⌘/Ctrl), Tooltip
