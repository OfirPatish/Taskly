# Taskly

A focused todo app — React, TypeScript, Chakra UI.

## Quick start

```bash
npm install
npm run dev
```

## Stack

React 19 · TypeScript · Vite 7 · Chakra UI v3 · Motion · next-themes

## Shortcuts

| Shortcut | Action |
| -------- | ------ |
| `⌘K` / `Ctrl+K` | Focus add input |
| `Enter` | Add task / Save edit |
| `Shift+Enter` | New line in task text |

Keybinds show **⌘** on Mac, **Ctrl** on Windows/Linux.
